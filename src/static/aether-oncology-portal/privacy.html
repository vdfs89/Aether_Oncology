Política de Privacidade

Última atualização: maio de 2026

⚠ Esta plataforma é uma ferramenta de apoio à decisão clínica para fins de pesquisa. Não constitui diagnóstico médico final.

1. Dados Coletados

A Aether Oncology processa exclusivamente os parâmetros biomecânicos de biópsia FNA inseridos pelo usuário profissional de saúde para fins de análise preditiva. Nenhum dado de identificação pessoal do paciente é coletado ou armazenado.

  • 30 parâmetros morfométricos nucleares (valores numéricos anônimos)
  • Timestamp da requisição (para auditoria MLOps)
  • Resultado da predição e feature de maior impacto

2. Uso dos Dados

Os dados processados são utilizados para:

  • Geração do diagnóstico preditivo via modelo Aether V2.0
  • Monitoramento de drift estatístico (MLOps)
  • Auditoria de uso da plataforma (logs/audit_trail.jsonl)

3. Retenção

Os registros de auditoria são mantidos temporariamente para fins de monitoramento do modelo. Não há compartilhamento com terceiros.

4. Base Legal (LGPD)

O tratamento dos dados é realizado com base no Art. 7º, II (pesquisa) e Art. 11, II, b (saúde — proteção da vida) da Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

5. Contato

Para dúvidas sobre esta política, acesse a documentação da API ou abra uma issue no repositório GitHub.
